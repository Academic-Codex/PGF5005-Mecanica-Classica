Pndulo simples com Euler simpltico (EulerCromer)

Neste caderno implementamos, de forma modularizada e comentada, o mtodo Euler simpltico para o pndulo simples adimensional com $$ H(q,p) = \frac{p^2}{2} - \cos q. $$

Do texto (seo 2.2), para Hamiltonianas separveis (H(q,p)=K(p)+U(q)), o Euler simpltico : $$ \boxed{ \begin{aligned} q^{(n+1)} &= q^{(n)} + \Delta t\,\frac{\partial H}{\partial p}\big|_{p^{(n)}} = q^{(n)} + \Delta t\,p^{(n)},\\[2pt] p^{(n+1)} &= p^{(n)} - \Delta t\,\frac{\partial H}{\partial q}\big|_{q^{(n+1)}} = p^{(n)} - \Delta t\,\sin\!\big(q^{(n+1)}\big). \end{aligned}} $$

No item 4.3, vamos gerar, para dois conjuntos de condies iniciais (librar e rotacionar), os grficos $q(t)\times t$ e $p(t)\times t$ usando $\Delta t \in \{10^{-1},10^{-2},10^{-3}\}$ as trs curvas no mesmo grfico para comparar.

No item 4.4, verificamos numericamente a energia total do sistema, isto , traamos $E(t)\times t$ para as mesmas condies e passos de tempo.

2) Modelo do pndulo e funes auxiliares

  • $\dot q = \partial H/\partial p = p$
  • $\dot p = -\,\partial H/\partial q = -\sin q$
  • $H(q,p) = \tfrac{1}{2}p^2 - \cos q$

A separatriz (limite entre librar e rotacionar) ocorre na energia $H=+1$.

In[2]:
def dq_dt(q, p):
    """dq/dt para o pndulo simples: p."""
    return p

def dU_dq(q):
    """derivada de U(q) = -cos(q): dU/dq = sin(q)."""
    return math.sin(q)

def H(q, p):
    """Hamiltoniana H = p^2/2 - cos(q)."""
    return 0.5*p*p - math.cos(q)

def classify_energy(q, p, eps=1e-9):
    """Classifica o regime pela energia: libration (-1 < H < 1) ou rotation (H > 1)."""
    E = H(q, p)
    if E > 1 + eps:
        return "rotation", E
    elif E < 1 - eps:
        return "libration", E
    return "separatrix", E

3) Um passo de Euler simpltico (EulerCromer)

Seguindo a ordem q-avana com $p^{(n)}$ e p-avana com $q^{(n+1)}$, preservando o carter simpltico do mapa numrico.

In[3]:
def euler_symplectic_step(q, p, dt):
    # 1) avana posio usando p^n
    q_next = q + dt * dq_dt(q, p)  # = q + dt * p
    # 2) avana momento usando q^{n+1}
    p_next = p - dt * dU_dq(q_next)  # = p - dt * sin(q^{n+1})
    return q_next, p_next

4) Integrador no intervalo ([0,T])

Armazenamos ((t,q,p,E)) para traar os grficos de (q(t)), (p(t)) e (E(t)).

In[4]:
def integrar_symplectic(q0, p0, dt, T):
    n_steps = int(np.ceil(T/dt))
    t = np.empty(n_steps+1)
    q = np.empty(n_steps+1)
    p = np.empty(n_steps+1)
    E = np.empty(n_steps+1)
    # estado inicial
    t[0] = 0.0
    q[0] = q0
    p[0] = p0
    E[0] = H(q0, p0)
    # iterao temporal
    for n in range(n_steps):
        q[n+1], p[n+1] = euler_symplectic_step(q[n], p[n], dt)
        t[n+1] = (n+1)*dt
        E[n+1] = H(q[n+1], p[n+1])
    return t, q, p, E

def plot_series_same_axes(series_list, xlab, ylab, title):
    """series_list: lista de tuplas (x, y, label)."""
    plt.figure()
    for x, y, label in series_list:
        plt.plot(x, y, label=label, lw=1.2)

    plt.ylabel(ylab)
    plt.title(title)

    modo, E0 = classify_energy(ic['q0'], ic['p0'])

8) Notas para interpretao dos resultados (para o relatrio)

  • Com o Euler simpltico, a energia no exatamente constante, mas tende a ficar quase-conservada, oscilando em torno de um valor mdio sem drift sistemtico para $\Delta t$ suficientemente pequeno.
  • Para a condio librar, $q(t)$ aproximadamente peridico e $p(t)$ alterna sinal.
  • Para a condio rotacionar, $q(t)$ cresce (ou decresce) monotonicamente no tempo (o pndulo d voltas) e $p(t)$ oscila em torno de valores diferentes de zero.
  • Reduzindo $\Delta t$ (de $10^{-1}$ para $10^{-3}$), as curvas convergem e as oscilaes de $E(t)$ diminuem.
  • Se desejar, voc pode embrulhar o ngulo com q_wrapped = ( (q+pi) % (2*pi) ) - pi apenas para visualizao; aqui mantivemos $q(t)$ cru, como solicitado.
