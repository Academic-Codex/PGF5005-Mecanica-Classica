Mtodo de Euler aplicado ao pndulo simples

Neste caderno implementamos, o mtodo de **Euler ** para resolver as equaes de Hamilton do pndulo simples adimensional, baseado nas equaes e notao do enunciado.

Hamiltoniana adimensional:
$$ H(q,p) = \frac{p^2}{2} - \cos q. $$

Equaes de Hamilton:
$$ \dot q = \frac{\partial H}{\partial p} = p,\qquad \dot p = -\,\frac{\partial H}{\partial q} = -\sin q. $$

**Mtodo de Euler ** conforme a Eq. (4) do texto:
$$ \begin{aligned} q^{(n+1)} &= q^{(n)} + \Delta t\,\frac{\partial H}{\partial p}\big|_{(q^{(n)},p^{(n)})} = q^{(n)} + \Delta t\,p^{(n)},\\ p^{(n+1)} &= p^{(n)} - \Delta t\,\frac{\partial H}{\partial q}\big|_{(q^{(n)},p^{(n)})} = p^{(n)} - \Delta t\,\sin\!\big(q^{(n)}\big). \end{aligned} $$

Observao: o item 4.2 pede Euler convencional. O item 2.2 do texto discute o Euler simpltico (EulerCromer), que ser usado depois (item 4.3). Aqui focamos apenas no Euler convencional para cumprir 4.2.

Campos de derivadas e energia

Codificamos $\dot q$ e $\dot p$ a partir da Hamiltoniana, e a energia $H(q,p)$ para checagens de consistncia (embora o mtodo de Euler no conserve energia).

def dq_dt(q, p):
    return p

def dp_dt(q, p):
    return -math.sin(q)

def H(q, p):
    return 0.5*p*p - math.cos(q)

Um passo do mtodo de Euler

Implementa exatamente as frmulas acima para avanar um passo $\Delta t$.

In[4]:
def euler_step(q, p, dt):
    q_next = q + dt * dq_dt(q, p)
    p_next = p - dt * (math.sin(q))  # equivale a p + dt * dp_dt(q,p)
    return q_next, p_next

Integrao no intervalo $[0, T]$

Guardamos o histrico de $(t,q,p,H)$ para facilitar grficos e diagnsticos.

In[5]:
def integrar_euler(q0, p0, dt, T):
    n_steps = int(np.ceil(T/dt))
    t = np.empty(n_steps+1)
    q = np.empty(n_steps+1)
    p = np.empty(n_steps+1)
    E = np.empty(n_steps+1)
    # estado inicial
    t[0] = 0.0
    q[0] = q0
    p[0] = p0
    E[0] = H(q0, p0)
    # lao temporal
    for n in range(n_steps):
        qn, pn = q[n], p[n]
        q[n+1], p[n+1] = euler_step(qn, pn, dt)
        t[n+1] = (n+1)*dt
        E[n+1] = H(q[n+1], p[n+1])
    return t, q, p, E
